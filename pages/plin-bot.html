Plin

Bot de Gestão Financeira & Self-Hosting

Python SQLAlchemy Docker Telegram API CasaOS

O Objetivo

O Plin foi desenvolvido para eliminar a fricção no registro de despesas diárias. Ao invés de abrir planilhas ou apps pesados, o usuário registra gastos e receitas através de uma conversa simples no Telegram, garantindo que o controle financeiro aconteça no momento em que a transação ocorre.

Engenharia e Infraestrutura

O projeto demonstra a aplicação de conceitos de backend robusto e infraestrutura autônoma:

  • Persistência Eficiente: Utiliza SQLAlchemy para abstrair a camada de dados, operando sobre um banco SQLite que garante portabilidade e rapidez.
  • Arquitetura de Comandos: Implementação de handlers assíncronos para lidar com múltiplas requisições simultâneas via python-telegram-bot.
  • Deploy e Autonomia: O bot é totalmente containerizado com Docker e hospedado no meu próprio servidor CasaOS, garantindo 100% de controle sobre os dados e disponibilidade.

O Futuro: Inteligência Artificial

Atualmente em desenvolvimento ativo, o Plin está evoluindo para além dos comandos fixos. O roadmap do projeto inclui:

  1. Processamento de Linguagem Natural (NLP): Implementação de IA para entender mensagens como "gastei 40 reais com pizza hoje" sem necessidade de comandos rígidos.
  2. Interface de Voz: Futura integração para processamento de áudios, permitindo o registro de transações por fala.
  3. Visualização: Geração de gráficos automáticos de saúde financeira diretamente no chat.
